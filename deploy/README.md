# nvssynthesis.github.io

Website showingcasing my personal projects. 

## Setup

### 0. clone the repo
```bash
git clone https://github.com/nvssynthesis/nvssynthesis.github.io.git
```

### 1. install dependencies
```bash
npm install
```

### 2. build typeScript
```bash
npm run build
```

### 3. open the website
Open `index.html` in your browser or use a local server like Live Server.

## Development

### TypeScript development
- source files live in `assets/js/src/`
- output files are compiled to `assets/js/dist/`

### watch mode (auto-compile on changes)
```bash
npm run dev
```

### one-time build
```bash
npm run build
```
